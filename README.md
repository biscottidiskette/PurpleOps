# PurpleOps

**PurpleOps** is an open-source platform for tracking red team attacks and blue team detections in order to identify detection gaps during purple team exercises.

This project is in **early development (pre-alpha)**.  
Follow the repo for updates as the MVP takes shape over the coming weeks.

## ğŸš€ Purpose

Modern purple team workflows often rely on spreadsheets or clunky tooling.  
PurpleOps aims to provide a clean, fast, and usable way to:

- Log red team actions and mapped MITRE ATT&CK techniques  
- Track blue team detections linked to those actions  
- Identify gaps in detection coverage  
- Produce actionable insights during exercises  

The goal: **Make purple team collaboration simpler, clearer, and more measurable.**
